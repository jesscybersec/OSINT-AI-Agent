# Project Overview

## Vision

Build a Kali Linux-friendly OSINT AI agent that orchestrates multiple open-source tools through a controlled, reviewable workflow and produces clean bilingual reports.

## Inspiration

This project is partially inspired by the OWASP SocialOSINTAgent model, especially its focus on:

- structured data aggregation
- social-platform intelligence
- AI-assisted synthesis
- Docker-based portability
- ethical collection through documented sources

## Problems This Project Solves

- OSINT results are scattered across tools and formats
- social and identity pivots are often done manually
- technical and human-focused OSINT are rarely unified
- final reporting is usually inconsistent and time-consuming

## Project Response

The agent follows a staged workflow:

1. Target intake
2. Scope and legality guardrails
3. Source-family selection
4. Collection
5. Normalization
6. Correlation
7. Analyst review
8. Bilingual report generation

## Scope

The intended scope includes:

- domains and subdomains
- IPs and organizational infrastructure
- usernames and aliases
- person names
- phone-related public OSINT
- company names and public registries
- social media and community platforms

## Intended Users

- cybersecurity students
- SOC analysts
- threat hunters
- authorized pentesters
- OSINT practitioners

## Positioning

This project sits between:

- an OSINT collection framework
- an AI-assisted analysis pipeline
- a bilingual reporting engine

The AI layer should help prioritize, summarize, correlate, and draft. It should not replace human judgment or legal boundaries.
